Keypoint data fetching:

SIFT (Scale-Invariant Feature Transform)
  - Scale-space extrema detection
    - Find local maxima across scale and space in image
  - Keypoint localization
    - Found potential keypoint locations are refined
    - Keeps strong interest points
  - Orientation assignment
    - Each keypoint is assigned an orientation
  - Keypoint descriptor
    - 16x16 neighbourhood for each keypoint is taken
    - Divided into 16 4x4 sub-blocks and matched an orientation

Brute-Force Matcher
  - Takes two sets of descriptors
  - Returns k-nearest neighbors (k = 2) descriptor set given by euclidean distance

Additionally a Ratio Test is applied as explained by D. Lowe in his paper.


Point Cloud Reconstruction:

- RANSAC for estimating fundamental matrix for processed keypoint descriptors
  - Then essential matrix is calculated from filtered points
  - Recover relative camera rotation and translation from estimated essential matrix
  - Compute ideal point coordinates from observed point coordinates
  - Reconstruct points by triangulations
  - Only points in positive depth (z > 0) and not outliners (z > 500) are kept
  - Points are then converted into point cloud structure containing points and color information
    - Surface normals are estimated
  - Point outliers are filtered if further away from neighbors in average
  - Points are aligned to ground thruth data using iterative closest point (ICP) algorithm
