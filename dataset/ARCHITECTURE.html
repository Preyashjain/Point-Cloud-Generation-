📊 Complete Pipeline - 6 Stages

📹 Stage 1: Frame Extraction
  • Decode video stream (MP4 format)
  • Extract 50 frames (every 3rd frame)
  • Resolution: 3840×2160 (4K)
  • Apply undistortion correction
⏱️ Time: 1-2 seconds
▼
🔍 Stage 2: SIFT Feature Detection
  • Scale-Invariant Feature Transform
  • Detect keypoints with orientation
  • Extract 128-dimensional descriptors
  • Output: ~2,000-5,000 points per frame
⏱️ Time: 20-30 seconds | Total: 100,000+ keypoints
▼
🔗 Stage 3: Feature Matching
  • Brute-force descriptor matching
  • Lowe's ratio test (0.7 threshold)
  • RANSAC outlier rejection
  • Output: 120,000+ high-quality matches
⏱️ Time: 5-10 seconds | Quality: >95% inliers
▼
🎯 Stage 4: 3D Reconstruction (Triangulation)
  • Essential Matrix via RANSAC 8-point algorithm
  • Camera intrinsic calibration
  • Direct Linear Transform (DLT) triangulation
  • Output: 100,000+ 3D points + colors
⏱️ Time: 10-20 seconds
▼
🧹 Stage 5: Filtering & Optimization
  • Statistical outlier rejection (20 neighbors)
  • Voxel grid filtering (1cm³ voxels)
  • Preserve spatial distribution
  • Result: ~83,000 high-quality points
⏱️ Time: 5-10 seconds | Quality: 30-40% noise reduction
▼
⚙️ Stage 6: Alignment & Validation
  • Point Cloud to Point Cloud ICP alignment
  • 4x4 transformation matrix computation
  • Chamfer, Hausdorff, F-Score metrics
  • Validation against laser-scanned ground truth
⏱️ Time: 5-10 seconds
